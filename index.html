Local Music Organizer for Mac

Clean up large local Music.app libraries safely.

Private beta

Local Music Organizer is currently in private beta. It is designed for Mac users with local Music.app libraries.

What it finds

  • Duplicate entries inside playlists
  • Duplicate Music library records pointing to the same local file
  • Missing local files
  • Missing artwork
  • Low bitrate MP3/AAC tracks
  • Playlist usage and compare results

Safety first

The app scans first, shows results, exports reports, and only applies changes when you explicitly choose an action. It does not physically delete audio files.

Requirements

Requires macOS 14 Sonoma or later. Apple Silicon and Intel builds are planned.

Status

Not available for public download yet.
